# Application to manage a car park interview task

How to use this project
------------------------
- Please clone this repository to your local machine.
- Compie the solution and run TdsCarParking.WebApi project.
- You can use Swagger UI to test the API endpoints. 
- Application use In-Memory database, so no additional setup is required.
- On application startup, the car park is seeded with 5 parking spaces, and required charges.
- In faivor of time integgration tests are not included, but can be added if required.
